## Тестовое "Реализация BMPN-процесса"

### Решения:
1. Принял решение не использовать bpmn-систему, её нет в требованиях а без неё реализация задачи проще(если нужно, могу добавить)
1. возвращаю результат согласно постановке, сами найденные записи вывожу в консоль
1. решение оформил в виде dsl, так как по моему мнению это наиболее декомпозируемый и расширяемый вариант

### Общий алгоритм работы
```mermaid
graph LR
    Запросить данные --> Отфильтровать по активности
    Отфильтровать по активности --> отфильтровать ИП
    отфильтровать ИП --> отфильтровать по нескольким идентификаторам региона
    отфильтровать по нескольким идентификаторам региона -> посчитать количество 
```

### Как запустить и проверить
1. скачать/склонировать
1. выполнить `mvn spring-boot:run` из корневой директории проекта (нужны java и maven)
1. открыть ссылку http://localhost:8080/swagger-ui.html
1. вызвать метод `/main/` с нужными параметрами, по дефолту - подставится номер пермского края

кроме того `IntegrationTest` при сборке прогоняет процесс

написал юниты и один интеграционный тест, покрытие можно посмотреть в идее
